# TaskNest

TaskNest is a productivity-focused web app for managing tasks, rewards, and study sessions.

## Features
- Add, complete, and manage tasks
- Earn points and unlock themes
- Timer and Pomodoro support
- Custom music recommendations

## Technologies
- HTML, CSS, JavaScript
- MongoDB (or your backend stack)

## How to Run
Open `index.html` in your browser or deploy it online.
